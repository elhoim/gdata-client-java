Google Base data API Overview

Get Started with the Google Base API

The Google Base data API is designed to enable developers to do two things:

  • Query Google Base data to create applications and mashups.
  • Input and manage Google Base items programmatically.

To get started with the Google Base data API:

  1. First, visit the API Demo Page. You can experiment with the API and see examples of what you can do with it.
  2. Read the Getting Started Guide, which contains important background information on the API, including how authentication works in Google Base.
  3. Read the Attributes and Queries chapter to learn how to query for data and attributes.

(For more information about Google Base, refer to the Google Base homepage.)
