Attributes Feed

The read-only feed provides data about how an item type has been described in Google Base and lists what values have been used most frequently for its attributes. This can be very helpful as you define new items that you wish to upload. This feed can show you how others have defined the item in existing entries, which can be useful models for your own items. See Attributes and Queries for some code samples on how to access this feed through the Google Base client library.

Feed URLs

You can get data about all item types, for a single item type, or for a single attribute of an item. The basic format is as follows: 

http://www.google.com/base/feeds/attributes?bq=[query-string]

If you want to get data about all of the items, you can leave off the ?bq=[query-string] component, as follows: 

http://www.google.com/base/feeds/attributes

Path parameters

In addition to the standard query parameters and standard path components, you can use the following parameters with the attributes feed. You can also run batch queries on the attributes feed.

Parameter Meaning Notes
bq is a query string that conforms to the Query Language Specification. Required. See the Attributes and Queries document for details on query syntax.
max-values [Optional] The number of distinct values returned in each entry. Most frequent N values along with their counts are returned. Default value: 0 for queries.

If you do not use this parameter, you only get high-level information about an item type.

Use this URL to get high-level statistics for each attribute defined for the digital camera item type. Currently max-values is set to 0 as a default, so detailed data is not provided.

http://www.google.com/base/feeds/attributes?bq=digital+camera

You can also use this URL to get the high-level statistics for the Jobs item type:

http://www.google.com/base/feeds/attributes/-/jobs

Only the first 25 items are shown by default. To get more items, increase the max-results value in the URL, like this:

http://www.google.com/base/feeds/attributes/-/jobs?max-results=30

Use this URL to get the 10 most frequently used values for each attribute defined for the Jobs item type:

http://www.google.com/base/feeds/attributes?bq=[item type:jobs]&max-values=10

Elements

Google Base uses the following elements in the Attributes feed:

Element Meaning
gm:attributes A list of attributes.
gm:attribute The attribute for which values are being displayed.
gm:value The value for each of the most frequently used values for this attribute.

Note that these elements use the gm: namespace. This namespace contains the metadata attributes that are used in the item types and attributes feeds. No other feeds use them and they are read-only, so users do not need to use this gm: namespace to input the attributes. The only impact on users is that you will see an extra line in the response, giving the location of the namespace, xmlns:gm ='http://base.google.com/ns-metadata/1.0.
