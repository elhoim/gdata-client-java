Featured Project Showcase

Select a language: | Java | JavaScript | Perl | PHP | Python | C# | VB.NET | ColdFusion

A picture is worth a thousand words, and sample applications speak volumes. The featured projects that we showcase on these pages should give you some inspiration for the myriad of applications that Google Base makes possible.

When you're ready to start building your own application, click on the programming language of your choice to find some example code.

 

BaseEstate.com is a Real Estate and Property search engine portal that uses Google Base and Google Maps.

verkoops.com is a dynamic web page built from the attributes feed that shows snippet results.

Fatdoor.com: Get to know your neighbors.

Honda Certified Used Cars allows users to search vehicles posted to Google Base.



Safarri.com lets you browse classifieds in Google Base. It overlays query results on Google Earth.

The Google Base Gadget is a simple and configurable interface for searching Google Base data from your Google Personalized Homepage or, using a Syndicated Google Base Gadget, almost any web page or blog.




Yahoo! Pipes is an interactive feed aggregator and manipulator.

LetMeDo Rental Listings provides a search interface to UK housing listings in Google Base.

Gumiyo is an end-to-end mobile commerce platform, connecting live buyers and live sellers through mobile phones, email and the web.

SeatSnapper searches for event tickets for sale on eBay that are listed in Google Base.

Find a convertible car 4sale lets you search for convertible cars listed for sale in Google Base, and plots them on Google Maps.

Job Central is a national non-profit labor exchange. They use the Google Base data API to include job listings from Google Base in their search results.

Agent Earth displays real estate listings from Google Base in Google Earth.

Warmoth has a search interface for Google Base that uses ColdFusion to search for products that they've uploaded. It sends a query to Google Base (appending a string that is present in only their item descriptions) and parses the returned XML into a page of product links, descriptions, and thumbnail images. You can find the code here.

Google Base Store Connector is a utility that enables online sellers to connect their eBay Stores, Amazon zShops and Marketplaces, and Yahoo! Stores to Google Base so searchers can find their items for sale on Google. It demonstrates Programmatic Login.

Google Blogoscoped is a simple search engine that uses the real PHP5 native XML capabilities. The link to the PHP5 source code is on the page.
