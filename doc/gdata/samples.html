Google Data APIs Samples

Contents

  1. Java samples

Java samples

These samples can be found in the downloadable Java client library package. For more information, see the Java client library developer's guide, or download the Java client library.

gdata/java/sample/authsub
GData AuthSub proxy, implemented using Java servlets; sample JavaScript client that accesses data through the proxy.
gdata/java/sample/calendar
Simple client that performs several hardwired manipulations on a Google Calendar.
gdata/java/sample/codesarch
Command line client to query the Google Code Search data API.
gdata/java/sample/spreadsheet/ImportClient.java
Command line client to import delimited text into a Google Spreadsheets worksheet.
gdata/java/sample/spreadsheet/IndexClient.java
Command line client to list metadata about the spreadsheets belonging to a user.
gdata/java/sample/spreadsheet/cell
Command line client to query and manipulate a spreadsheet using a cells feed.
gdata/java/sample/spreadsheet/gui
GUI application that displays part of a spreadsheet in a window on your desktop.
gdata/java/samples/spreadsheet/list
Command line client to query and manipulate a spreadsheet using a list feed.
gdata/java/samples/spreadsheet/tester
Simple client that performs several hardwired manipulations on a GData feed (typically Blogger).
