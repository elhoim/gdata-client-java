Getting Started

We recommend that you sign up for a Google Apps domain account as a sandbox to develop and test within the proper context. See the table below for the appropriate Google Apps edition for the API(s) you are looking to use. If you do not already have a Google Apps domain account that you can use as a development sandbox, sign up for one by clicking one of the buttons below. Once you have an account, dive into the API documentation and get started.
